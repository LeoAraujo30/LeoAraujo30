  <img height="20" alt="GIF" src="https://github.com/LeoAraujo30/LeoAraujo30/blob/main/gifs/rocket.gif"/> **About Me:**
  <div align="justify">
I've always wanted to learn programming but never knew exactly where to start. However, at the end of 2021, I discovered Trybe and its JavaScript web development course. In January 2022, I began my application process, and two months later, I joined my first class. 
My studies started with the basics: Git for version control, HTML, CSS, pure JavaScript, and testing with Jest. Then, I moved on to Front-end development, where I learned to build web pages with React, manage global states with Redux or Context and Hooks, and create tests with React-Testing-Library. 
When I got to the Back-end, I explored the concept of containerization with Docker, worked with relational and non-relational databases using MySQL and MongoDB, built REST APIs (in some cases with TypeScript and OOP) in Node.js using Express with Sequelize or Mongoose, and developed tests with Mocha and Chai. 
Later on, I delved a bit into computer science, switching my programming language to Python. I ran some tests with Pytest, studied a bit about web scraping, algorithms, and types of data structures. 
Throughout this entire learning period, I had the opportunity to develop personal and group projects, applying agile methodologies like Scrum and Kanban. Finally, in May 2023, I completed my training in Web Development.

  <img height="20" alt="GIF" src="https://github.com/LeoAraujo30/LeoAraujo30/blob/main/gifs/rocket.gif"/> **Sobre Mim:**
  <div align="justify">
Sempre tive vontade de aprender programação, mas nunca soube exatamente por onde começar. Porém, no final de 2021, eu descobri a Trybe e seu curso de desenvolvimento web em JavaScript. Em janeiro de 2022, iniciei meu processo seletivo, e dois meses depois, entrei na minha primeira turma.
Meus estudos começaram pelo básico: Git para controle de versionamento, HTML, CSS, JavaScript puro e testes com Jest. Em seguida, avancei para o Front-end, onde aprendi a desenvolver páginas web com React, gerenciar estados globais com Redux ou Context e Hooks, além de criar testes com React-Testing-Library.
Ao chegar no Back-end, explorei o conceito de conteinerização com Docker, trabalhei com bancos de dados relacionais e não relacionais utilizando MySQL e MongoDB, construí APIs REST (em alguns casos com TypeScript e POO) em Node.js usando Express com Sequelize ou Mongoose, e desenvolvi teste com Mocha e Chai.
Posteriormente, me aprofundei um pouco em ciência da computação, mudando minha linguagem de programação para Python. Realizei alguns testes com Pytest, estudei um pouco sobre raspagem de dados, algoritmos e tipos de estruturas de dados.
Durante todo esse período de aprendizado, tive a oportunidade de desenvolver projetos pessoais e em grupo, aplicando metodologias ágeis como Scrum e Kanban. E finalmente, em maio de 2023, concluí minha formação em Desenvolvimento Web. 
  </div>
